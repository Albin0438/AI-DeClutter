# AI-DeClutter 🛡️

An optimized, automated, and unified filter list built for browser extensions like **uBlock Origin** and **AdGuard** to strip away AI clutter, synthetic content farms, and forced "Ask AI" interface elements.

## Features
- 🚀 **Zero Maintenance:** Updated automatically every single day via GitHub Actions.
- 🎯 **Performance Focused:** Deduplicates and optimizes community rules to keep your browser fast.
- 🧹 **Pure Web Experience:** Hides forced AI summary boxes, search overviews, and blocks known AI scraping/generation sites.

## How to Use

1. Copy the following URL:
`https://raw.githubusercontent.com/YOUR_USERNAME/AI-DeClutter/main/ai-clean-filters.txt`
*(Replace YOUR_USERNAME with your GitHub username)*

2. Open **uBlock Origin Dashboard** -> **Filter lists**.
3. Scroll to the bottom, check **Import**, paste the link, and click **Apply changes**.
